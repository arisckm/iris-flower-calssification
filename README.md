A simple yet powerful **Machine Learning** app that classifies iris flowers into three species:
- Setosa
- Versicolor
- Virginica

This app uses the **Iris dataset** from scikit-learn and a **Logistic Regression** model to predict species based on:
- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)
